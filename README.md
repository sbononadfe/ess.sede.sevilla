# ESS – Sede Sevilla
 
Sitio web informativo de la sede de ESS en Sevilla, desarrollado como página estática y desplegado mediante GitHub Pages.  
Este repositorio contiene el código fuente de la web, así como los recursos necesarios (imágenes, estilos y scripts) para su correcto funcionamiento.
 
## Descripción
 
El proyecto muestra información relacionada con la sede de ESS en Sevilla: contexto, servicios, localización y otros contenidos relevantes para personas interesadas en la entidad.  
Su objetivo principal es ofrecer una interfaz clara y sencilla que facilite la consulta de la información desde cualquier dispositivo.
 
## Características principales
 
- Diseño estático, sencillo de mantener y desplegar.
- Estructura pensada para navegación rápida por secciones.
- Maquetación responsive para su correcta visualización en ordenadores, tablets y móviles.
- Recursos estáticos (CSS, imágenes, fuentes, etc.) organizados en carpetas separadas.
 
## Tecnologías utilizadas
 
- **HTML5** para la estructura del contenido.
- **CSS3** para el diseño y la maquetación.
- **JavaScript** (opcional) para pequeñas interacciones o mejoras de usabilidad.
- **GitHub Pages** como plataforma de despliegue.
 
## Objetivo Final
Este proyecto se ha realizado para representar una de las mejores y avanzadas sedes de enseñanzas superiores.
